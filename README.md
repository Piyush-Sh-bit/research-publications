## 📚 Publications & Pipelines

### 1. [Scale, Strategy, and Structure: A Multilevel Meta-Analysis of MLLMs](./2026_mllm_meta_analysis)
* **Status:** Under Review (2026)
* **Code Folder:** [`/2026_mllm_meta_analysis`](./2026_mllm_meta_analysis)
* **Description:** A rigorous two-level mixed-effects meta-analysis evaluating the performance of 21 Multimodal Large Language Models (MLLMs) across 7 visual reasoning benchmarks. Calculates standardized effect sizes, tests publication bias, maps efficiency Pareto frontiers, and executes leave-one-out sensitivity analyses.
* **Tech Stack:** Python, Pandas, Statsmodels, SciPy

## 🚀 How to Use This Repository

Each publication is contained entirely within its own folder to prevent dependency conflicts. To run the code for a specific paper:

1. Clone this repository:
   ```bash
   git clone https://github.com/YourUsername/YourRepositoryName.git
   cd YourRepositoryName
   ```

2. Navigate to the specific publication's folder:
   ```bash
   cd 2026_mllm_meta_analysis
   ```

3. Follow the specific `README.md` present inside that folder for specialized installation and execution instructions (such as `requirements.txt`).
